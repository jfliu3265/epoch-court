# 万象净界：纪元王庭

一个原生 Web 的幻想冒险垂直切片。玩家通过四个小游戏净化两个界域、重建王庭，并解锁会改变旧地图规则的时代透镜。

## 运行

```bash
npm install
npm run dev
```

打开终端显示的本地地址。生产构建：

```bash
npm run build
npm run preview
```

## 验证

```bash
npm run check
npm run test:e2e
```

## 操作

- WASD / 方向键：移动
- 空格 / Enter：行动
- Shift：冲刺
- E：次行动或重置
- Q：切换时代透镜
- Escape：暂停
- 触屏和标准手柄也可使用

## 内容

- 单色无声序章
- 纪元王庭 Hub 与档案馆建设
- 原初荒原、森语庭院
- 史莱姆闪避场、遗迹推箱、花粉连线、萤火节奏
- 像素层与水彩层两种时代透镜
- 自动存档、导入导出、PWA、低画质与无障碍设置
